About CodeBuddy

CodeBuddy was created by the Piccolo lab. Our goal is to enable people around the world to more effectively learn how to write code and analyze data. With this Web-based tool, instructors create programming and data-analysis exercises that are organized into courses, assignments, and exercises. For each exercise, learners are presented with a description of the exercise, as well as the expected output. The learners must then write code to match the expected output. These outputs can be either text-based or graphics-based.

Our GitHub site provides a list of key features that CodeBuddy provides. We welcome bug reports and feature requests as well as pull requests. Before attempting to add new features, we encourage you to contact us.
