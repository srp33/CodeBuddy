Allow students to download their code for this course?
With this option, students are allowed to download their code for all assignments and exercises in this course as a single HTML file. The file includes their latest, passing submission for each exercise. To provide context, assignment introductions and exercise instructions are provided. Timed assignments, hidden assignments, and non-code exercises are excluded.

Passcode (optional):
Students are required to enter this code in order to register for the course.

Virtual Buddy configuration (optional):
This feature is currently experimental. When it is enabled, you can configure assignments in the course to help students when they are stuck on programming exercises. It uses the Chat Completions endpoint of OpenAI's API (similar to what is used by ChatGPT). To enable this feature, you must specify configuration information in YAML format, as shown in this example. Please note that you must indicate whether this feature is enabled in the settings for each assignment.
