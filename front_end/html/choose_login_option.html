Privacy disclosure

This website uses Internet cookies to store information in your browser. When you log in to this website, it will store your user ID so that you do not need to log in each time you visit a page on the website. As you are logging in, it will store a temporary cookie so that you can be redirected back to the page you were attempting to access. After you have logged in, this website will use cookies to store information about which parts of the website you are allowed to access as well as content from courses, assignments, and exercises. By storing this information in cookies, the website can increase its performance because it does not need to access its database as often. After you log in, this website will store your name, login ID, and email address in its database. Finally, this website will keep a log of which parts of the website you access and how long it takes to access them. This information provides insights to the website administrators so they can seek to improve the website's performance and analyze usage. This logging information is not stored in cookies but rather in text files on the web server. Only website administrators are allowed to access this information.

By clicking the "Accept" button, you agree to the storing of cookies on your device and information about you on this website. At any time, you may delete your account on this website. Doing so will delete all cookies specific to this website from your browser. It will also delete your information from the database used by this website. You will not be able to recover this information.
